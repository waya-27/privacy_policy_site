Handling of Data

  1. Storage of Input Content
    Content entered by the user is not saved in the app. It is temporarily sent to the API but is discarded after processing.
  2. Anonymous Use
    You can use the app without logging in. We do not collect personal information.
  3. Usage Restrictions
    Slander and posts containing personal names are prohibited. Inappropriate content will be automatically filtered.
  4. About AI Generated Text
    Text generated by AI is not intended to attack or slander specific individuals. Users themselves should not use it for the purpose of hurting others.
