1. Information We Collect

Information collected

  • Product data (name, price, stock status)
  • Category information
  • Search content for recipe suggestions (ingredients, keywords, etc.)
  • Voice input data (only when using voice recognition; temporary data for processing purposes)
  • Device information such as advertising identifiers
  • App usage status (screen transitions, frequency of feature usage, etc.)
  • App crash information and error logs

Method of collection

  • Direct input by the user
  • Automatic collection by the app (SDKs for analysis, advertising, crash analysis, etc.)

2. Purpose of Use

  • Provision and improvement of inventory management and recipe suggestion functions
  • Analysis for service improvement (UI/UX improvement, bug fixes, etc.)
  • Ad delivery and effectiveness measurement
  • Support and inquiry response

3. Sharing and Provision of Information

Provision to third parties

In principle, we do not provide personal information to third parties.

Exceptional provision

  • When required by law
  • When there is user consent

4. Information Protection

Security measures

  • Data encryption
  • Appropriate access control
  • Periodic security audits

5. Voice Recording Permissions

We may use the device's "Microphone (Voice Recording)" permission for the voice input function during recipe suggestions.

  • Voice data is used only for recipe searching.
  • Voice is immediately deleted after conversion to text and is not saved or transmitted externally.
  • This permission is not available unless the user uses the voice input function.
